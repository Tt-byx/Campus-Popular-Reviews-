# 校圈 —— fk!肉蛋gay好胖

> 一个学生共建的校园点评社区，像大众点评 + 虎扑一样，评课、评食堂、评社团、评活动。

## 🌟 核心功能
- 📚 **课程 & 教师评价**：打标签（#给分好 #作业多）、看雷达图
- 🍜 **食堂档口反馈**：实时评分、排队情况、上传实拍
- 🎯 **社团/活动聚合**：招新信息、成员真实体验
- 🔒 **学号验证 + 匿名发布**：真实可信，保护隐私

## 🛠 技术栈（计划）
- 前端：Vue 3 + Element Plus
- 后端：Spring Boot（Java）
- 数据库：MySQL
- 项目管理：TAPD
- 部署：阿里云学生机
